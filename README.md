# Spam SMS Detection

This project classifies SMS messages as Spam or Ham using Machine Learning and Natural Language Processing.

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* TF-IDF Vectorization
* Multinomial Naive Bayes

## Workflow

1. Load SMS dataset
2. Preprocess text
3. Convert text using TF-IDF
4. Train Naive Bayes model
5. Predict Spam/Ham messages
6. Evaluate model accuracy
